Model's Performance Comparison on Animal Detection Data

Selected Models:

InceptionV3

A deep CNN architecture from Google that uses factorized convolutions and inception modules for improved accuracy and reduced computational cost.

DenseNet

Each layer is connected to every other layer (dense connectivity), promoting feature reuse, better gradients, and fewer parameters.

VGG19

A deep but simple network using stacked 3x3 convolutions. Though heavier, it remains a strong baseline in many vision tasks.

Similarities & Differences

Similarities

  • All are CNN architectures pretrained on ImageNet
  • Used for transfer learning in image classification
  • Capable of capturing animal features like fur, ears, shapes

Differences

  • InceptionV3: Lightweight with modular design
  • DenseNet: Dense connections and fewer parameters
  • VGG19: Straightforward but heavier and slower

Why Use These for Animal Detection?

  • Transfer Learning: Saves time and resources
  • High Accuracy: InceptionV3 and DenseNet outperform with limited data
  • Strong Visual Feature Extraction: Good at patterns like fur, tails, textures
